# Andrew Pryde's dotfiles

A not particularly well thought through collection of dotfiles.

## Tools of the trade

 - [zsh][1]
 - [zplug][2]
 - [tmux][3]
 - [vim][4]
 - [`rcm`][7]

## Installation

 1. Install [`rcm`][7]
 2. `make up`
